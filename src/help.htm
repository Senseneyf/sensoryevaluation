User Manual for Sensory Evaluation Website


Connecting to the Website

To connect to the website, you need to be connected to the Cisco AnyConnect Secure Mobility Client VPN. Use your CWU username and password to connect to this. If you have any trouble connecting to the VPN contact the help desk. After connecting to the VPN enter the Sensory Evaluation IP address into an internet browser.

Signing In

When entering the Sensory Evaluation page, you will see a screen (figure 1) asking for your user name and password. Enter your user name and password and press the continue button. It you cannot remember your user name of password contact your teacher for assistance.

Basic Use

After signing in the list of tests available to you will appear (figure 2). Click on the title of any test to allow judges to use that test. Click the “Edit” button (located to the right of every test) to edit the test.  Click the “Delete” button (located to the right of every test) to delete the test.  In the righthand corner click the “Logout” button to logout of the website. To create a new test, click either the “Create Test” button in the top left hand corner or the “CREATE NEW TEST” button at the bottom of the page. To return to this main page at any time click the “My Tests” button in the top left-hand corner.

Creating Tests

Once you click the create test button you will be taken to a page where you will be asked to choose which type of test you will be creating (figure 3). After choosing one of the three options for type of test you will be taken to one of two slightly different looking forms where you will need to input the information you want for that test. If you choose “Duo/Trio Test” or “Triangle Test” you will be taken to a form that requires you input a test name, test description and test product (Figure 4). Test name should describe exactly what you are testing. The test description should contain all information that the judge will require to take the test. This can include what is being tested, descriptive variables, etc. The product name should indicate what kind of food or drink the sample you are giving users is. If you choose “Intensity Test” you will be taken to a form that requires you input a test name, test description, test product, number of samples and scale type (Figure 5). Test name, test description and product name will be similar for intensity tests as for duo/trio and triangle tests. Number of samples is the number of samples you will be giving the judge to try not including any reference samples (or in other words only samples that you want the judge to enter information for). The scale type has options for “9 point scale”, “6 point scale”, and “unstructured scale”. This means whether you would rather have the judge input a number or indicate a point on a line to indicate their perception of the different samples. Clicking “Edit” on the “My Tests” page will return you to this page and allow you to change the original answers you gave for a test.

Having Judges use the Tests

First, to look at test results, download test results and prepare for having judges use the test click on a test name on the “My Tests” page (Figure 6). To download the test results, click the “Download” Button. To see the random ids associated with samples for a test click the “PREP TEST FOR JUDGMENT” button (Figure 7). From this page you can print the ids with the “PRINT” button or click “BACK” to return to the Test Results page. On the sample id page, you will see a table with headers of Sample 1, Sample 2, etc. this indicates the order in which samples should be presented. The letter before the random three-digit id indicates the sample version. So, if B.202 is located beneath Sample 1 this means that sample version B should be allocated the id 461 and presented to the judge first. If you click the “JUDGEMENT” button on the Test Results page you will open a page that allows the user to enter data for the test. These pages can be vastly different depending on the type of test and the type of scale.
